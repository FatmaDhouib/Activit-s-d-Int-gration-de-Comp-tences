Q2 — A jouter une méthode age() à l'entité Etudiant
Indication  : La méthode doit calculer dynamiquement l'âge de l'étudiant à partir de son
dateNaissance (de type LocalDate) et de la date du jour. Utilisez java.time.Period ou
ChronoUnit.YEARS. Avec Lombok, pensez à annoter correctement la classe pour éviter les
conflits avec les getters générés.
public int age() { 
    return Period.between(this.dateNaissance, LocalDate.now()).getYears(); 
}

Q13 — A jouter Redis pour activer le cache
Indication  : Ajoutez Redis comme couche de cache sur les endpoints de lecture. Utilisez
spring-boot-starter-data-redis et activez le cache avec @EnableCaching dans votre
classe de configuration. Annotez la méthode de service avec @Cacheable et invalidez le
cache avec @CacheEvict sur les opérations d'écriture. Ajoutez un conteneur Redis dans
votre docker-compose.yml.
@Cacheable(value = "etudiants") 
public List<EtudiantDTO> findAll() { ... } 
@CacheEvict(value = "etudiants", allEntries = true) 
public EtudiantDTO save(EtudiantDTO dto) { ... }
